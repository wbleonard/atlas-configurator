# Delete a Cluster
Deletes a cluster.

**URI** : `/atlas/deleteCluster/?clusterName=:cn`

**Method:**: `DELETE`

**Auth required**: Yes

Authorization: [Stitch User ID](http://stitch-sdks.s3-website-us-east-1.amazonaws.com/stitch-sdks/js/4/interfaces/stitchuser.html#id)

## Success Response

**Code**: `200 OK`

**Response example**

```json
"BrianDevCluster3 deleting..."
```
